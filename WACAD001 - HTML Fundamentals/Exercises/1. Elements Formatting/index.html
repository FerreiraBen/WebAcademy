História do HTML!


O HTML é uma linguagem de marcação e não uma linguagem de programação rsrs para descrição de documentos web.

Foi criada para publicação de conteúdos na web, podendo conter texto, imagem, vídeo, áudio entre outras formas de representação de conteúdo.

Os primeiros dias da web, tem havido muitas versões do HTML: HTML, HTML 2.0, HTML3.0, HTML4.01, XHTML e HTML5 (UAU).


É de finalidade do navegador de internet ler documentos HTML e exibi-los.

A declaração DOCTYPE ajuda o navegador a exibir uma página web de forma decente corretamente.

A declaração doctype não diferencia MAIÚSCULAS de minúsculas.

Todos casos são aceitáveis.

O HTML5 não requer que elementos vazios sejam fechados. Mas se você quiser uma validação rigorosa, ou você precisa fazer o seu documento legível por analisadores de XML, você deve fechar todos os elementos HTML.
